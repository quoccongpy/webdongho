Một web thương mại điện tử sử dụng mô hình n-tiers, Design Patterns : repository và unit of work
Có các chức năng : thêm,xoá, sửa category,cover type,brand,product. Giỏ hàng và đang tìm hiểu thanh toán bằng momo

_ Entity Framework : codefrist
_ mô hình MVC,
_ mô hình n-tiers, Design Patterns : repository và unit of work
